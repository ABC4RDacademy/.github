# ABC4RD Academy

ABC4RD Academy is an open-source Bitcoin education initiative helping students, developers, and researchers learn Bitcoin development through public curricula, curated resources, practical workshops, and respectful contribution to the global Bitcoin ecosystem.

## Mission

Our mission is to help new developers, students, and researchers build a strong technical foundation in Bitcoin and become useful, respectful participants in open-source Bitcoin communities.

## What We Focus On

- Bitcoin fundamentals and protocol literacy
- Bitcoin development and open-source contribution
- Lightning Network learning paths
- Wallets, self-custody, and practical security
- Privacy, cryptography, and responsible research
- Curated public resources and conference notes
- Student-friendly contribution practice

## For Students

Start with the curriculum, follow the reading list, and use the onboarding guide to learn how to read repositories, open issues, and prepare small documentation contributions.

## For Contributors

We welcome improvements to explanations, source links, glossary entries, examples, and beginner-friendly learning paths. Please keep contributions accurate, well-sourced, and respectful of upstream maintainers.

## For Experts and Speakers

If your public work is referenced here, we welcome corrections and accuracy feedback. We do not treat public references as endorsement or affiliation.

## Open-Source Principles

- Create useful materials before asking for attention.
- Cite primary public sources whenever possible.
- Separate verified information from material that requires verification.
- Respect maintainers, speakers, authors, and communities.
- Prefer small, practical contributions over promotional activity.
