# Contributing to ABC4RD Academy

Thank you for helping improve ABC4RD Academy.

## Contribution Principles

- Be accurate before being loud.
- Prefer primary public sources.
- Separate verified information from `requires verification`.
- Do not imply endorsement, partnership, or affiliation.
- Keep explanations beginner-friendly.
- Respect maintainers, authors, speakers, organizers, and contributors.

## Good First Contributions

- Fix typos or broken links.
- Add a primary source.
- Improve a glossary entry.
- Clarify a beginner explanation.
- Suggest a small curriculum exercise.
- Mark unverified information as `requires verification`.

## Before Opening a Pull Request

1. Read the repository README.
2. Check existing issues and pull requests.
3. Keep the change small and specific.
4. Add source links when changing factual claims.
5. Explain why the change helps learners.

## Source Verification

Use these labels in notes and tables:

- `verified` - confirmed by an official source.
- `secondary source` - credible but not primary.
- `requires verification` - not confirmed yet.
- `not used` - excluded because the source is unclear, private, spam-like, or unreliable.
